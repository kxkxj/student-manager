# 学生成绩管理系统

基于 C++11 的控制台学生成绩管理程序，支持多课程成绩管理、排序、统计与排名。

## 功能

- **添加学生** — 录入学号、姓名和任意多门课程成绩
- **删除学生** — 按学号删除
- **修改成绩** — 按学号和课程修改分数
- **查询学生** — 显示单个学生的所有成绩
- **显示全部** — 列出所有学生及成绩
- **课程统计** — 统计某门课的平均分、最高分、最低分
- **成绩排序** — 按某门课成绩从高到低排名
- **不及格名单** — 列出某门课不及格的学生
- **成绩排名** — 按总分或平均分排名，并标记比课程最多的学生少修了哪些课
- **保存/加载** — 数据持久化到 `save.txt`

## 构建

### 依赖

- 支持 C++11 的编译器（MSVC / GCC / Clang）
- CMake >= 3.10

### 编译

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

### 运行

```bash
# Windows
main.exe

# Linux / macOS
./main
```

## 文件结构

```
.
├── CMakeLists.txt         # CMake 构建配置
├── main.cpp               # 程序入口与菜单
├── StudentManager.h/.cpp  # StudentManager 类实现
├── class.h/.cpp           # Student 类定义与实现
├── utils.h                # 输入工具函数
├── save.txt               # 数据持久化文件
└── README.md
```

## 数据格式

`save.txt` 每行一条学生记录，格式为：

```
学号 姓名 课程1:分数1 课程2:分数2 ...
```
